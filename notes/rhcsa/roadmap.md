# RHCSA Study Timeline

## Weeks 1–2 — Foundation
- File system hierarchy
- File permissions (chmod, chown, ACLs)
- Users and groups
- Package management (dnf)
- Basic shell scripting
- Cron and at jobs
- SSH configuration
- Boot process overview

## Weeks 3–4 — Storage & systemd
- systemd units and targets
- journalctl and logging
- Partitioning (fdisk, parted)
- LVM (pvcreate, vgcreate, lvcreate)
- Stratis storage
- VDO volumes
- firewalld
- NetworkManager and network config
- Hostname and DNS

## Weeks 5–6 — SELinux & containers
- SELinux modes and policies
- chcon and restorecon
- audit2allow and audit logs
- semanage (ports, fcontexts, booleans)
- Podman containers
- autofs
- NFS mounts
- Time sync with chrony
- Kernel parameters (sysctl)
- Boot target recovery and root password reset

## Weeks 7–8 — Exam simulation
- Full timed mock exams
- Break-fix scenarios
- Root password reset drills
- Weak spot identification and repeat
- Speed drills on common tasks
